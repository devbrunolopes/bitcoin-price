# bitcoin-price
Aplicativo para consultar o preço atual de um Bitcoin em diversos câmbios.

Criei esse aplicativo para consolidar o conhecimento adquirido em:

- Delegate
- Protocols
- Extensions
- API
- Networking
- JSON
